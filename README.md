# LifeClicker: The OOP Journey ⏳
**Final Project for Object-Oriented Programming Course**

## 📖 รายละเอียดโปรเจกต์
LifeClicker เป็นเกมแนว Incremental/Clicker ที่จำลองการเดินทางของชีวิตมนุษย์ โดยทุกการคลิกคือ "1 ปี" ที่ผ่านไป ผู้เล่นต้องบริหารจัดการ "Life Energy" เพื่อซื้อการอัปเกรดสถานะ (Upgrades) และเผชิญกับเหตุการณ์ต่าง ๆ ตามช่วงอายุ (Life Events) โดยเป้าหมายคือการใช้ชีวิตให้คุ้มค่าที่สุดก่อนหมดอายุขัย

## 🛠 Framework & Principles
- **Language:** Python
- **Library:** Pygame
- **Principles:** OOP (Inheritance, Polymorphism, Encapsulation, Abstraction) และ SOLID Principles
- **Design Patterns:** Strategy Pattern, Factory Pattern
